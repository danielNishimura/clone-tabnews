# Gerando documentação de componentes

Sugestões para gerar documentação automática de componentes React:

1. Storybook — ótimo para documentação visual e testes interativos.

- Instalar: `npx sb init`
- Rodar: `npm run storybook`

2. react-docgen + styleguidist — extrai props e gera páginas estáticas.

3. TypeDoc (para TypeScript) — gera API docs a partir de tipos e comentários.

Escolha um destes e eu posso ajudar a integrar ao projeto.
